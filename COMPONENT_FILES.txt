LAYOUT COMPONENTS (4 total)
===========================

1. Sidebar.tsx
   - Collapsible sidebar navigation (64px collapsed, 256px expanded)
   - 7 navigation items: Dashboard, Inbox, Cases, Constituents, Templates, Knowledge Base, Reports
   - City logo/name display at top
   - Toggle button at bottom
   - Active state styling (blue-600 background)
   - Tooltip hints in collapsed mode
   - Dark mode support
   - Responsive collapse on mobile
   - Uses lucide-react icons
   - Features: useState hook, usePathname for active states, useCallback for toggle

2. Header.tsx
   - Sticky top header (z-50)
   - Page title on left
   - Controls on right:
     * NotificationBell widget
     * Dark mode toggle (Moon/Sun icons)
     * User profile dropdown (Avatar + dropdown menu)
   - User dropdown options: Profile, Notifications, Sign out
   - Responsive layout
   - Uses next-themes for dark mode
   - Props: title, userName, userEmail, userAvatar, callbacks

3. NotificationBell.tsx
   - Bell icon with red unread badge
   - Dropdown menu with notifications
   - Features:
     * Unread count display (9+)
     * Notification list with timestamps
     * Read/unread state indicators
     * "Mark all as read" link
     * Individual notification dismiss buttons
     * Time formatting (just now, 1m ago, 1h ago, 1d ago)
   - Customizable notification interface
   - onClick and onDismiss callbacks

4. DashboardLayout.tsx
   - Complete dashboard wrapper combining Sidebar + Header
   - Flex layout with proper spacing
   - Sidebar left (fixed positioning)
   - Main content area with Header + scrollable content
   - Props: title, children, user details, city info, callbacks
   - Ready-to-use for page layouts

USAGE PATTERNS
==============

1. Import and use in page/component:

   import { Button, Card, CardContent, CardHeader, CardTitle } from '@/components/ui'
   import { DashboardLayout } from '@/components/layout'

   export default function MyPage() {
     return (
       <DashboardLayout title="My Page" userName="John">
         <Card>
           <CardHeader>
             <CardTitle>Title</CardTitle>
           </CardHeader>
           <CardContent>
             <Button>Click me</Button>
           </CardContent>
         </Card>
       </DashboardLayout>
     )
   }

2. TypeScript with component props:

   import type { ButtonProps } from '@/components/ui'

   function MyButton(props: ButtonProps & { custom?: string }) {
     return <Button {...props} />
   }

3. Dark mode toggle:

   - Header component includes Moon/Sun toggle
   - Uses next-themes useTheme hook
   - Automatically managed
